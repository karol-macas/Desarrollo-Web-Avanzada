Desarrollo de Aplicación Web con ASP .NET Core

-Descripcion-
Aplicación Web de ASP.NET Core MVC
Lenguaje de programación C♯
Base de datos en SQL Server
Scaffold-DbContext
EntityFramework para crear vistas y procesos CRUD automáticos

-Instalacion-
Para ejecutar este proyecto debes crear la base de datos o importar la base de datos con el nombre BDCRUD y cambiar la cadena de conexión del archivo appsettings.json
Por ejemplo:
"ConnectionStrings": {
  "cadenaSQL": "Server=MSI\\SQLEXPRESS1 ;DataBase=DBCRUD;Integrated Security=true"
}, 

-Resultados-
Una vez hecho todo el proceso pones Ejecutar o correr y te abrira el navegador
y te mostrare la Pagina Web
